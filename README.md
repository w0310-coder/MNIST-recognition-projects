# MNIST-recognition-projects
This repository has open-sourced 5 MNIST handwritten digit recognition projects.
MNIST手写数字识别项目合集
一、项目说明
本合集包含 5个迭代优化版本 的MNIST手写数字识别项目，基于 Python + PyTorch 搭建神经网络模型，实现0-9手写数字的智能分类与精准识别。
二、文件结构说明
总仓库内包含5个独立子文件夹，分别对应模型迭代升级、逐步优化的不同版本；
每个子文件夹内，统一包含3类配套文件：
1.  .py  源码文件：可直接本地运行的完整项目源代码
2.  .txt  日志文件：程序运行全过程日志、模型测试准确率、性能结果等数据
3.  .pdf  文档文件：完整代码排版整理文档，方便快速查阅、学习与参考
三、运行使用说明
1. 本地提前部署好 Python、PyTorch 相关运行环境
2. 进入对应版本的项目文件夹，直接运行  .py  源代码文件即可启动程序
3. 程序运行时会自动在线下载MNIST官方数据集，全程无需手动额外配置，开箱即用
